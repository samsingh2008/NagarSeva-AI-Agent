## Development Patterns Established

### Frontend Pattern
```
Component → Hook (useApi) → Axios → Backend
                    ↓
              State Update
                    ↓
              Re-render UI
```

### Backend Pattern
```
Route → Controller → Service → Model → Database
                         ↓
                   Response
```

### AI Integration Pattern
```
Request → Service.analyzeWithGemini() → Prompt
                    ↓
              Gemini API Call
                    ↓
              Parse & Return Data
```

## Common Questions Answered

**Q: Where do I create a new page?**  
A: `client/app/(routes)/my-page/page.tsx`

**Q: Where do I add an API endpoint?**  
A: Create controller in `server/src/controllers/`, route in `server/src/routes/`

**Q: How do I use Gemini AI?**  
A: Import from `server/src/config/gemini.ts` in your service

**Q: Where do I add a MongoDB model?**  
A: Create in `server/src/models/` using Mongoose

**Q: How do I style components?**  
A: Use Tailwind CSS classes directly in JSX

**Q: Where do I put helper functions?**  
A: Frontend helpers in `client/utils/`, backend in `server/src/utils/`
